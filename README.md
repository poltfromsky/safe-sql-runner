# Safe SQL Runner

Скрипт для безопасного выполнения SQL-запросов к базе данных PostgreSQL из командной строки. Разрешены только запросы SELECT, запросы на изменение данных (INSERT, UPDATE, DELETE, DROP и другие) блокируются. Если в запросе не указан LIMIT, он добавляется автоматически.

## Как запустить

1. Клонировать репозиторий

2. В папке проекта зайдите в  файл с именем .env.example,скопируйте содержимое в .env  и после равно укажите в нем данные для подключения к своей базе данных PostgreSQL:
   
   DB_HOST=
   DB_PORT=
   DB_NAME=
   DB_USER=
   DB_PASSWORD=
   Что означает каждая строка:

   - DB_HOST — адрес сервера базы данных. Если PostgreSQL установлен на вашем компьютере, оставьте localhost.
   - DB_PORT — порт, на котором работает PostgreSQL. По умолчанию используется 5432, менять не нужно, если вы не задавали другой порт при установке.
   - DB_NAME — имя базы данных, к которой будет подключаться скрипт. Эту базу нужно создать заранее, например через pgAdmin: правой кнопкой на Databases → Create → Database, и указать любое имя.
   - DB_USER — имя пользователя PostgreSQL. Обычно это postgres, он создается автоматически при установке PostgreSQL.
   - DB_PASSWORD — пароль, который вы задавали при установке PostgreSQL для пользователя postgres.

   Все значения указывайте без кавычек, каждое на отдельной строке.
   Не забудьте сохранить файл!

3. Установите библиотеки из requirements.txt (в папке  проекта по свободному месту правой кнопкой мыши→открыть в терминале):
   
  pip install -r requirements.txt (если не получается то можно через команду python main.py "SELECT * FROM students")
 

4. Запустите скрипт(с помощью терминала в папке проекта):
   
   python main.py
